# 必死チェッカーdroid mod chmate　連携
- 必死チェッカーdroidの　mod版 (改造版)
- chmateと連携できます。
- 元のAPKからJADXでエクスポートした後、できる限りOSSに近づけるよう整理・修正されたものです。読める状態に近づけています。（完全なOSSではありません。）

# 変更点
- フルスクリーン表示
- ID表記を大きく
- モダンテーマ
- AMOLED Black テーマ
- モダンなコンテキストメニュー
- カレンダーデザインの変更
- ⋮ から URLコピー機能 や全レスコピー など
- Android16対応
- 同梱されていた mediba/OpenX 広告SDKのソースコードを削除。(完全に削除できているかどうかは保証しません。)
- 画面下部の広告レイアウト、およびアプリ側の MasAdView 関連ライフサイクル処理を削除。
- ACCESS_NETWORK_STATE 権限を削除（広告SDKでのみ使用されていたため）
- ターゲットSDK更新時に白いプラットフォームテーマへフォールバックしないよう、明示的なアプリテーマを追加

- 本アプリはhissi.orgがSSL/TLSに対応していない関係上、一部通信にHTTP（非暗号化通信）を使用しています。 Android 9（API 28）以降の端末では平文通信がデフォルトで制限されるため、android:usesCleartextTraffic="true" を有効化しています。
